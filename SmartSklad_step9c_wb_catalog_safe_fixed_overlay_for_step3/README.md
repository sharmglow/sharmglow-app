# Смарт Склад / SmartSklad

Десктопное приложение для учёта товаров на складе с поддержкой маркетплейсов WB и OZON.

## Быстрый старт

### Вариант 1 — запуск из исходников

```bash
# 1. Установить Python 3.11+ с python.org (если нет)

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить
python sharmglow.py
```

### Вариант 2 — собрать .exe (Windows)

```bash
pip install -r requirements.txt
build_windows.bat
```

Готовый файл будет в `dist\SmartSklad.exe`.  
База данных `sharmglow.db` создаётся рядом с `.exe` при первом запуске.

## Возможности

| Вкладка | Что делает |
|---|---|
| 🔍 Сканер | Сканирование штрихкода → приход или перемещение. Автоматически определяет WB/OZON по складу назначения и подставляет штрихкод |
| 📊 Остатки | Таблица остатков: Приход / →WB / →OZON / →др. / Остаток ТВ-2 |
| 📦 Товары | CRUD справочника. Два штрихкода на товар: WB и OZON |
| 📋 История | Журнал прихода и перемещений с фильтрами |
| 🏭 Склады | CRUD справочника складов. Если название содержит «WB» — подставляется штрихкод WB |
| 🏷️ Этикетки | Очередь печати + **редактор дизайна** этикеток с превью |

## Редактор этикеток

На вкладке «Этикетки» → «✏️ Редактор дизайна»:

- **Размер** — любой в мм (58×40, 40×30, 58×30…)
- **Фон** — выбор цвета
- **Рамка** — вкл/выкл
- **Каждый блок** настраивается отдельно:
  - Показать/скрыть (чекбокс)
  - Позиция X/Y в % от размера этикетки
  - Размер шрифта, жирный, курсив
  - Цвет текста
  - Выравнивание (лево/центр/право)
  - Шрифт (sans/serif/mono)
- **Блоки:** Значок WB/OZON, Название, Разделитель, Артикул, Логотип, Штрихкод, Цифры штрихкода, Цена

Можно создавать несколько шаблонов (для WB, OZON, внутренние этикетки и т.д.)

Печать → PDF, открывается в любом PDF-просмотрщике, оттуда на принтер.

## Импорт из старого xlsx

Файл → Импорт из xlsx — читает `skladuchet_v2.xlsx` (листы 📦 Товары, 🏭 Склады, ➕ Приход, 🔄 Перемещения).

## База данных

`sharmglow.db` — SQLite файл. Создаётся автоматически при первом запуске рядом с программой.  
Резервная копия — просто скопируйте `sharmglow.db`.  
Для доступа с двух компьютеров — положите `sharmglow.db` на сетевой диск и укажите путь в `sharmglow.py`:

```python
db = DB(path=r"\\server\share\sharmglow.db")
```
